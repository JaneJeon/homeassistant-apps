# Home Assistant App: grafana-image-renderer app

## How to use

This app really does nothing. It is just an grafana-image-renderer.

When started it will print the configured message or "Hello world" in the log.

It will also print "All done!" in `/share/grafana-image-renderer_app_output.txt` to show
simple grafana-image-renderer of the usage of `map` in app config.
